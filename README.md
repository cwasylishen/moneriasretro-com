# moneriasretro.com

Tienda online de juegos retro, papelería con aroma y muñecas de vestir. Pedidos por WhatsApp, envíos a todo Costa Rica.

## Stack

Sitio estático HTML/CSS. Sin build. Hosted en Cloudflare Pages.

- `index.html` — homepage con catálogo completo
- `blog/` — notas y artículos
- `assets/styles.css` — sistema de diseño
- `assets/images/` — fotos de productos
- `_headers` / `_redirects` — config Cloudflare Pages
- `sitemap.xml`, `robots.txt`

## Deploy

Conectar este repo a Cloudflare Pages. Output directory: `.`.
